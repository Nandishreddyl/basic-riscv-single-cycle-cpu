# Single Cycle RISC-V Processor in Verilog

## Overview

This project implements a basic Single Cycle RISC-V Processor in Verilog HDL.

The processor was developed as a learning project to understand Computer Architecture and CPU Design before implementing a customizable pipelined RISC-V processor.

## Implemented Modules

- Program Counter (PC)
- Instruction Memory
- Instruction Decoder
- Control Unit
- Register File
- Arithmetic Logic Unit (ALU)
- Data Memory
- Write Back Unit
- CPU Top Integration

## Supported Instructions

- ADD
- SUB
- AND
- OR
- XOR

## Tools Used

- VS Code
- Icarus Verilog
- GTKWave
